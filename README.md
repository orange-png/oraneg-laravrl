# orange/orange-laravel


这里是对于框架的介绍

# 对于框架的 要求

Laravel >= 5.1 & PHP > 7.1.3 

# 安装的方式

```yml
composer require "OrangePng/orange-laravel"
```



解释路由
```php
<?php
Route::get('/', 'SJunitController@index');
Route::post('/', 'SJunitController@store')->name('junit.store');
?>
```
